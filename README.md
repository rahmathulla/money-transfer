# money-transfer
**A sample REST application For Money Transfer**

**Technologies Used**: 
1. Jersey
2. Embedded Jetty
3. Swagger
4. jOOQ
5. Java 8
6. Lombok Library
7. H2 Database
8. Flywaydb (for migration)
9. Guice
10. Maven

**How to Run?**:

1. Execute mvn clean install
2. Execute java -jar target\money-transfer-0.0.1-SNAPSHOT.jar (The application will run in Port 9091.)
3. Navigate to http://localhost:9091 to get Swagger UI.
4. On the URL box - paste http://localhost:9091/swagger.json and click Explore. All the Resources will be available for try!
